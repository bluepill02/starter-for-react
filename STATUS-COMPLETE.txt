QUICK START - DEPLOYMENT STEPS
───────────────────────────────────────────────────────────────────────

1. PREPARE ENVIRONMENT
   export APPWRITE_ENDPOINT=https://your-appwrite-server
   export APPWRITE_PROJECT_ID=your-project-id
   export APPWRITE_API_KEY=your-api-key
   export APPWRITE_DATABASE_ID=your-database-id

2. DEPLOY PHASES IN ORDER
   
   Phase 1 (Security):
   $ node scripts/deploy-phase1.js --production
   
   Phase 2 (Compliance):
   $ node scripts/deploy-phase2.js --production
   
   Phase 3A (Reliability):
   $ node scripts/deploy-phase3a.js --production
   
   Phase 3B (Deployment Safety):
   $ node scripts/deploy-phase3b.js --production
   
   Phase 3C (Monitoring):
   $ node scripts/deploy-phase3c.js --production

3. VALIDATE DEPLOYMENT
   $ curl https://your-functions/health
   $ curl https://your-functions/health/ready
   $ curl https://your-functions/metrics

4. SETUP KUBERNETES (OPTIONAL)
   $ kubectl apply -f infra/k8s/deployment.yaml
   $ kubectl apply -f infra/k8s/service.yaml

5. START MONITORING STACK
   $ docker-compose -f docker-compose.monitoring.yml up -d
   
   Access:
   - Prometheus: http://localhost:9090
   - Grafana: http://localhost:3000 (admin/admin)
   - Jaeger: http://localhost:16686
